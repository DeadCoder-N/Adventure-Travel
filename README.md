# Adventure-Travel

A comprehensive hotel booking and travel website featuring multiple homepage designs, accommodation listings, and travel services.

## Features

- **Multiple Homepage Designs**: 10 different homepage layouts including Hotel Reservations, Luxury Travel, Adventure Travel, Package Tours, Boutique Hotels, Mountain Tour, Safari Tour, Photography Tour, City Tour, and Island Tour
- **Accommodation System**: Complete hotel booking functionality with room details, pricing, and availability
- **Responsive Design**: Mobile-friendly layouts for all devices
- **Interactive Elements**: Video popups, image galleries, and booking forms
- **Service Pages**: About us, contact, services, and activity pages

## Getting Started

1. Open `Index.html` in your web browser to view the main homepage
2. Navigate through different homepage designs in the `Home-Pages/` folder
3. Explore accommodation options in the `accommodation/` folder
4. Browse additional pages and activities in the `Pages/` folder
